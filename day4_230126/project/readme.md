# Landing Page Clone – HTML & CSS

This project is a **static clone** of a modern landing page design inspired by:

👉 https://framebloxpages.framer.website/landing/06  

The goal of this project is **learning and practicing front-end fundamentals** by recreating a real-world UI using **pure HTML and CSS**, without frameworks or JavaScript.

---

## 📌 Project Objective

- Practice **HTML semantic structure**
- Strengthen **CSS fundamentals**
  - Box model
  - Positioning
  - Spacing & alignment
  - Typography
- Understand how modern landing pages are structured
- Translate a design into real code

This is **not a pixel-perfect clone**, but a **close structural and visual recreation** for learning purposes.

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- No JavaScript
- No frameworks (Bootstrap, Tailwind, etc.)
- No libraries
