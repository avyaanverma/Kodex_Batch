
# CSS Basics Notes (Without Position, Flexbox, Grid)

## 1. What is CSS?
CSS (Cascading Style Sheets) is used to **style HTML elements**.

HTML → structure  
CSS → appearance  
JavaScript → behavior

## 2. Ways to Apply CSS

### 2.1 Inline CSS (❌ avoid)
```html
<p style="color:red;">Hello</p>
````

### 2.2 Internal CSS

```html
<style>
  p {
    color: blue;
  }
</style>
```

### 2.3 External CSS (✅ best)

```html
<link rel="stylesheet" href="style.css">
```

---

## 3. CSS Syntax

```css
selector {
  property: value;
}
```

Example:

```css
p {
  color: red;
  font-size: 16px;
}
```

---

## 4. CSS Selectors

### 4.1 Element Selector

```css
p {
  color: blue;
}
```

### 4.2 Class Selector

```css
.text {
  color: green;
}
```

### 4.3 ID Selector

```css
#main {
  background-color: yellow;
}
```

⚠️ ID should be **unique**

---

### 4.4 Group Selector

```css
h1, h2, p {
  color: black;
}
```

---

### 4.5 Universal Selector

```css
* {
  margin: 0;
  padding: 0;
}
```

---

## 5. Colors in CSS

### 5.1 Named Colors

```css
color: red;
```

### 5.2 HEX

```css
color: #ff0000;
```

### 5.3 RGB / RGBA

```css
color: rgb(255, 0, 0);
color: rgba(255, 0, 0, 0.5);
```

---

## 6. Units

### Absolute

* `px`

### Relative

* `%`
* `em`
* `rem`
* `vh`
* `vw`

Example:

```css
div {
  width: 50%;
  height: 100vh;
}
```

## 9. Background Properties

```css

  background-size: cover;
```

## 10. Box Model (VERY IMPORTANT)

Every element has:

```
Content → Padding → Border → Margin
```

### Example:

```css
.box {
  width: 200px;
  padding: 20px;
  border: 2px solid black;
  margin: 10px;
}
```

---

## 11. Width & Height

```css
div {
  width: 300px;
  height: 150px;
  max-width: 500px;
  min-height: 100px;
}
```

---

## 12. Border

```css
div {
  border: 2px solid red;
  border-radius: 10px;
}
```

---

## 13. Margin & Padding

### Margin

```css
div {
  margin: 20px;
}
```

### Padding

```css
div {
  padding: 15px;
}
```

### Individual Sides

```css
margin-top: 10px;
padding-left: 20px;
```

---

## 14. Display Property

### Block

```css
display: block;
```

Examples:

* div
* p
* h1

---

### Inline

```css
display: inline;
```

Examples:

* span
* a

---

### Inline-Block (IMPORTANT)

```css
display: inline-block;
```

Allows:

* width
* height
* margin

---

### None

```css
display: none;
```

Removes element from layout.

---

## 15. Visibility

```css
visibility: hidden;
```

Element hidden but **space remains**.

---

## 16. Overflow

```css
div {
  overflow: hidden;
  overflow: scroll;
  overflow: auto;
}
```

Controls content overflow.

---

## 17. Opacity

```css
div {
  opacity: 0.5;
}
```

Range: `0` to `1`

---

## 18. Box Sizing

```css
* {
  box-sizing: border-box;
}
```

Includes padding & border in width/height.

---

## 19. Pseudo-classes

```css
a:hover {
  color: red;
}

input:focus {
  border: 2px solid blue;
}
```

Common:

* `:hover`
* `:active`
* `:focus`

---

## 20. Pseudo-elements

```css
p::first-letter {
  font-size: 30px;
}
```

Common:

* `::before`
* `::after`
* `::first-letter`

---

## 21. CSS Comments

```css
/* This is a comment */
```

---

## 22. Cascading & Priority (IMPORTANT)

Priority order:

1. Inline CSS
2. Internal CSS
3. External CSS
4. Browser default

### `!important` (avoid)

```css
p {
  color: red !important;
}
```


## 23. Common Beginner Mistakes

* Forgetting units (`px`)
* Using IDs everywhere
* Inline CSS abuse
* Ignoring box model
* Not using `box-sizing: border-box`
