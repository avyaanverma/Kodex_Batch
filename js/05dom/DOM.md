## HTML Document Object Model
The Document Object Model (DOM) is a programming interface (API) for HTML and XML documents that represents the page as a structured, hierarchical tree of nodes.

It allows scripting languages, such as JavaScript, to dynamically access, add, modify, or delete content, structure, and style in a web browser.
