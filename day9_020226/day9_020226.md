# pseudo selectors

HTML can be used to add text inside the webpage
but you can't do the same with css

Elements ke andar kuch extra cheezein add karne ke liyeh
this is the core of animations


## ::before and ::after

This content we are supposed to add in before and after property these are not actual text like in h1 as if you see you can't select them 
These are virtual elements , illusion of elements



```css
h1{
    color: #e62060; // must to give
}

h1::before{
    content: "HAHAHA";
    color: royalblue;
    font-size:200px;

}

```

Ab let's say aapko div banana ho

## ::hover wala ek example


left se right ya right se left 
```html
    <h1>Home</h1>
```

```css
    h4{
        margin: 20px;
        background: red;
        font-size:30px;
        width: fit-content;
        position:relative;
    }

    h4::after{
        content: "";
        width:0%;
        height:3px;
        background:white;
        position:absolute;
        left:0;
        bottom:0;
        transition: all ease 4s;
    }

    h4:hover::after{
        width:100%;
    }

```
